# SunsetFlow — אתר מידע על הפסטיבל

SunsetFlow הוא אתר שמציג מידע על פסטיבל SunsetFlow — אירוע חווייתי שמתקיים בזמן השקיעה, וכולל מוזיקה, אווירה, ואטרקציות מיוחדות.

## 📄 תוכן האתר
האתר כולל:
- דף בית עם מידע כללי על הפסטיבל
- פרטים על תאריך, מיקום ואופי האירוע
- מידע למבקרים

(ניתן להרחיב ולוסיף עמודים נוספים בהמשך, כמו לוח זמנים, אמנים, מפה, כרטיסים ועוד.)

## 🚀 איך לצפות באתר
כדי לצפות באתר, יש לפתוח את הקובץ `index.html` בדפדפן.

אם האתר פועל דרך GitHub Pages:
**קישור לאתר החי:**  
https://your-username.github.io/SunsetFlow/  
(להחליף בשם המשתמש שלך)
